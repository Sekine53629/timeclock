## 計算根拠

### 割増率

| 労働分類 | 割増率 | 根拠 |
|---------|-------|------|
| 時間外労働（持ち帰り） | 1.25倍 | 労働基準法第37条第1項 |
| 休日労働 | 1.5倍 | 労働基準法第37条第1項 |
| 深夜労働（22:00～5:00） | 1.35倍 | 労働基準法第37条第4項 |
| 休日+深夜 | 1.6倍 | 上記の組み合わせ |

### 作業時間推定方法

コード変更量（追加行数、削除行数、変更ファイル数）から実際の作業時間を推定。

**推定式:**
```
推定時間（分）= (追加行数/10)×5 + (削除行数/10)×2 + ファイル時間

ファイル時間 = 10分（1ファイル目） + 5分×(追加ファイル数)
```

### 除外プロジェクト

学習用プロジェクト（"practice"、"kadai"を含む名称）は集計対象外。
