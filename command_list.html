⏱️ 打刻システム - コマンドリスト

📌 エイリアス設定推奨：
alias tc='python /path/to/timeclock/cli.py' を設定すると、
python cli.py の代わりに tc で使えます。

🔵 打刻コマンド

コマンド 説明 使用例
start <account> <project> 作業開始 python cli.py start user1 myproject
break 休憩開始 python cli.py break
resume 休憩終了・作業再開 python cli.py resume
end 作業終了 python cli.py end
status 現在の状態表示 python cli.py status

📊 レポートコマンド

コマンド 説明 使用例
report daily <account> 日別レポート python cli.py report daily user1
report daily <account> --date <date> 特定日のレポート python cli.py report daily user1 --date 2025-10-10
report daily <account> -v 詳細表示 python cli.py report daily user1 -v
report project <account> <project> プロジェクト別レポート python cli.py report project user1 myproject
report project <account> <project> --start-date <date> --end-date <date> 期間指定レポート python cli.py report project user1 myproject --start-date 2025-10-01 --end-date 2025-10-31
report monthly <account> 月次レポート（今月） python cli.py report monthly user1
report monthly <account> <YYYY-MM> 特定月のレポート python cli.py report monthly user1 2025-09
report monthly <account> -v 詳細表示（日別内訳） python cli.py report monthly user1 -v
report monthly <account> -o <file> HTML出力（印刷可能） python cli.py report monthly user1 -o report.html

⚙️ 設定コマンド

コマンド 説明 使用例
setup 初期セットアップ
（Google Driveパス設定）		 python cli.py setup
config show <account> 設定を表示 python cli.py config show user1
config set <account> --closing-day <15|31> 締め日設定 python cli.py config set user1 --closing-day 15
config set <account> --standard-hours <hours> 標準労働時間設定 python cli.py config set user1 --standard-hours 8

📋 一覧コマンド

コマンド 説明 使用例
list accounts アカウント一覧 python cli.py list accounts
list projects <account> プロジェクト一覧 python cli.py list projects user1

💡 よく使うコマンド例

基本的な1日の流れ

タイミング コマンド 説明
朝（9:00） python cli.py start user1 project-alpha 作業開始
昼（12:00） python cli.py break 休憩開始
昼（13:00） python cli.py resume 休憩終了・作業再開
午後（15:00） python cli.py end
python cli.py start user1 project-beta		 別プロジェクトに切替
夕方（18:00） python cli.py end 作業終了
終業後 python cli.py report daily user1 今日の作業時間確認

月次レポート作成と印刷

コマンド 説明
python cli.py report monthly user1 -o monthly_report.html 月次レポートをHTML出力
open monthly_report.html (macOS)
start monthly_report.html (Windows)		 ブラウザで開いて印刷
🔧 トラブルシューティング：
• 作業を開始できない → python cli.py status で状態確認
• 既に作業中の場合 → python cli.py end で終了してから再開
• 休憩を開始できない → 先に start で作業を開始する必要があります
